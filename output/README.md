# Output Folder

This folder stores generated results. It is not the source code of the thesis workflow.

Typical subfolders are:

- `eda/` for exploratory analysis outputs,
- `experiments/` for clustering experiment outputs,
- `plots/` and `pdf/` for exported figures,
- `preprocessed/` for some generated preprocessed CSV exports, and
- `run/` or temporary folders for intermediate artifacts.

In a cleaned thesis-facing copy of the repository, this folder may be empty or contain only placeholders. That is expected.

The canonical scripts that generate these outputs are documented in the root [README](../README.md).
